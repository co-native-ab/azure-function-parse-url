# azure-function-parse-url

Example Azure Function to parse url into html and markdown using readability view.

**Please note**: This code is just an example and not intended for production usage.

## Local usage

Run the function locally:

```shell
nvm use --lts
npm install
npm start
```

Invoke the function:

```shell
curl -s --data '{"url": "https://choosealicense.com/licenses/mit/"}' http://localhost:7071/api/parse | jq
```
